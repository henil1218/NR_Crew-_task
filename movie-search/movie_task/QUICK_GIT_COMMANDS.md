## Initial Setup (One-time)

```bash
# Initialize git
git init

# Add all files
git add .

# First commit
git commit -m "Initial commit: Movie Search App"

# Add remote (replace YOUR_USERNAME and REPO_NAME)
git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git

# Push to GitHub
git branch -M main
git push -u origin main
```

## Daily Workflow

```bash
# Check status
git status

# Add changes
git add .

# Commit
git commit -m "Your commit message"

# Push
git push
```

## Useful Commands

```bash
# View commit history
git log

# View remote repositories
git remote -v

# Pull latest changes
git pull

# Create a new branch
git checkout -b feature-name

# Switch branches
git checkout main
```
